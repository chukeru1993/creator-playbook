# 贡献指南

欢迎提交能被多个创作者或视频项目复用的改进。

## 适合提交

- 新的 Agent Skill，或已有 Skill 的清晰输入、输出和检查机制；
- 已在真实项目验证过的视觉样式规范；
- 分镜语法、镜头卡字段和验收方法；
- 可跨项目使用、并能说明适用边界的模板。

## 不适合提交

- 单支视频的口播稿、时间码、封面、录屏、音视频或渲染结果；
- 私密项目资料、密钥、客户信息和未获授权的第三方素材；
- 没有来源说明的效果数据、工具承诺或经验结论。

## 提交前

1. 将新增资产放入职责对应的目录，并更新 docs/资产索引.md。
2. 说明它解决的重复问题、适用条件和不适用条件。
3. 保持项目专属事实在原项目中，不把它们写成通用规则。
4. 检查相对链接、模板占位符，以及 Skill 所需的附属文件。

一次 Pull Request 尽量只完成一个可以独立评审的改进。

## 许可证

提交 Pull Request 即表示你确认自己有权提交该内容，并同意该贡献以本仓库的 MIT License 发布。
